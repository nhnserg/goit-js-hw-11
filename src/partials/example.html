HTML-partials

The posthtml-include Плагин позволяет добавлять код из других файлов HTML (фрагменты) на HTML -страницу файлыФрагменты хранятся в папке src/partials .А Дополнение происходит при составлении файлов страниц, таких как index.html .

Этот файл example.html является примером такого фрагмента, с разметка определенной части главной страницы.Это очень удобно для Развитие команды по студенческим проектам, когда несколько человек пишут в одном HTML файл одновременно.

The syntax for adding a fragment to the main file is: <include src="./partials/filename.html"></include>
